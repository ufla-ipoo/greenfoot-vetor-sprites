TÍTULO DO PROJETO: exemplo-vetor-sprites-greenfoot
OBJETIVO DO PROJETO: Exemplo de uso de vetores e animação de sprites usando o Greenfoot
VERSÃO ou DATA: 2023.10.09
COMO INICIAR O PROJETO: Crie um objeto Cavaleiro, coloque-o no mundo, e clique em Executar.
AUTORES: Julio César Alves
INSTRUÇÕES PARA O USUÁRIO:
O cavaleiro pode caminhar para a direita e para a esquerda usando as setas do teclado.
O código mostra como fazer a animação do cavaleiro parado e caminhando para cada direção.
